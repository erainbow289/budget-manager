# budget-manager (Python)

## Overview
Budget Manager is a Python-based project designed to simulate a simple resource management system. It allows users to create budgets, track spending, and monitor remaining funds.

This project was built as part of my independent learning in software development and contributes to my progression into game development by strengthening my understanding of systems thinking, logic flow, and structured problem-solving.

---

## Purpose of the Project
The goal of this project was to:
- Develop strong Python programming fundamentals
- Practice building structured, object-oriented systems
- Improve logical thinking and problem decomposition
- Simulate a basic resource management system similar to those found in game economies

---

## Features
- Create and manage multiple budgets
- Track spending per budget
- Automatically calculate remaining funds
- Summarise total budget, spending, and remaining balance
- Input validation to prevent incorrect operations

---

## Skills Demonstrated
- Python programming (Object-Oriented Programming)
- Data structures (dictionaries and lists)
- System design and logical flow
- Error handling and validation
- Problem-solving and debugging
- Basic simulation of resource management systems

---

## What I Learned
This project helped me understand how to design and structure a system that tracks and updates values over time. It improved my ability to break down problems into smaller components and reinforced the importance of clean, maintainable code structure.

It also gave me insight into how resource systems work, which is directly relevant to game development concepts such as economy systems, inventory management, and player resource tracking.
